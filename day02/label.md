## label 예약어? 🤔

label은 Java에서 많이 사용하지 않는 예약어다. 두 개 이상의 for나 while 루프를 사용할 때, 바깥쪽 루프의 시작점으로 이동하기 위해 사용한다.

```java
Loop:
for(int i = 0; i < n; i++) {
    for(int j = 0; j < m; j++) {
        if(arr[i][j] == 1) break Loop;
    }
}
```
